# Simple Todo App

The Simple Todo App is a lightweight task manager for creating, completing, and deleting todos. It was built as a refactor-focused project to practice JavaScript modules and object-oriented programming.

## Functionality

The app renders an initial list of todos from JavaScript data. Users can open a popup form, create a new todo with an optional due date, mark todos as complete or incomplete, and delete todos from the list. New todos receive unique IDs so their checkboxes and labels stay correctly connected.

## Technology

This project uses semantic HTML, Flat BEM CSS, JavaScript modules, ES6 classes, DOM templates, form validation, and the uuid package for unique todo IDs.
